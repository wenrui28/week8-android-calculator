# Week 8 - First Android Application

This repository contains a simple Android calculator app created for the Week 8 Object-Oriented Programming exercise.

## Assignment requirements covered
- Built in Android Studio
- Java language
- Two input fields for numbers
- Four buttons: add, subtract, multiply, divide
- Result shown on screen
- Clear view IDs such as `inputFirstNumber`, `buttonPlus`, and `textResult`

## How to run
1. Open Android Studio.
2. Choose **Open** and select this project folder.
3. Let Gradle sync finish.
4. Run the app on an emulator or Android device.
5. Enter two numbers and press one of the operation buttons.

## Example test
- Input: 14 and 17
- `+` => Result: 31
- `-` => Result: -3
- `x` => Result: 238
- `/` => Result: 0.8235294117647058

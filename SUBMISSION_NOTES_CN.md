# Week 8 提交说明（中文）

## 作业要求总结
根据题目要求，需要提交两样东西：
1. 一个公开的 GitHub（或其他 public git）仓库链接，里面放解决方案代码。
2. 一个视频链接，视频中解释代码并演示程序运行。

## 这个项目已经满足的要求
- Android Studio 项目
- Java 语言
- 两个数字输入框
- 四个按钮：加、减、乘、除
- 屏幕上显示结果
- 使用了清晰的控件 id：
  - `inputFirstNumber`
  - `inputSecondNumber`
  - `buttonPlus`
  - `buttonMinus`
  - `buttonMultiply`
  - `buttonDivide`
  - `textResult`

## 你提交前需要做的事
1. 把这个文件夹上传到你自己的 GitHub 新仓库。
2. 仓库设为 Public。
3. 不要在截止时间后继续 commit。
4. 按照 `VIDEO_SCRIPT.md` 录一个 3-5 分钟视频。
5. 在 Moodle 提交：
   - 你的 GitHub 仓库链接
   - 你的视频链接

## GitHub 仓库名建议
`week8-android-calculator`

## GitHub 链接格式示例
`https://github.com/你的用户名/week8-android-calculator`
